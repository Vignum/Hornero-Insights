# Hornero Agri

Agricultural research datasets focused on crop stress, production reconciliation, weather context, and commodity fundamentals.

## Sample Files

- agri_crop_stress_sample.json
- agri_crop_stress_sample.xlsx

## What This Product Covers

- USDA versus local source reconciliation
- Production divergence analysis
- Crop stress monitoring
- Weather and precipitation context
- Soil moisture conditions
- Export demand indicators
- Agricultural market monitoring

## Data Sources

Examples include:

- USDA
- CONAB
- MAGyP
- Bolsa de Cereales
- BCR
- NOAA
- NASA POWER

## Research Principles

- Evidence before conclusions
- Source reconciliation over single-source narratives
- Explicit data quality reporting
- No invented values or unsupported estimates
